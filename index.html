Features
Everything you need, nothing you don't
No frameworks, no bloat. Pure vanilla JavaScript, a native C/GTK3 shell, and an AI brain that actually helps.
🔍

Smart Spotlight Search

Type anything. Unit conversions, time zones, morse code, tip calculator, color previews, dice rolls — 40+ instant answers with zero latency, no API key needed.

🧠

Dual-Process AI Brain

A fast local brain (Ollama) handles easy stuff instantly. A big brain (Claude API) only wakes up when the little one isn't sure. Gets smarter over time.

🪟

Real Window Management

Drag, snap, pin-on-top, mini-mode PiP, corner quadrants, grid tiling, cascade. Works like a real desktop, because it is one.

🌐

Multi-Tab Browser

Chromium on the ISO. In web mode, a full proxy-backed browser with tabs, bookmarks, history, and keyboard shortcuts (Ctrl+T, Ctrl+W, Ctrl+Tab).

🎯

Intent Kernel

Tell Astrion what you want, not what to do. "Create a folder called Projects and put a file in it" — the OS figures out the steps and runs them.

🔒

Safety by Design

4 capability tiers. AI can never promote itself. Dangerous actions preview before executing. Provenance on every artifact. Socratic prompts keep you thinking.

🎮

Games & Fun

Wordle, Sudoku, Chess, Snake, 2048, Tetris, Minesweeper, Reaction Test — plus a soundboard, emoji kitchen, and pixel art editor.

🎨

Customizable Everything

Accent colors, wallpapers, dock visibility, night shift, focus mode. Your OS, your rules.

💻

Native C/GTK3 Shell

The desktop shell is written in C. Panel, dock, window manager — all native. Ctrl+R does nothing. No browser chrome leaking through.
